Calculadora Web
Descripción

Este proyecto es una calculadora básica desarrollada con HTML, CSS y JavaScript. Permite realizar operaciones matemáticas simples de manera rápida e interactiva mediante una interfaz moderna y fácil de usar.

Características
Suma, resta, multiplicación y división.
Botón para borrar resultados.
Diseño responsivo.
Interfaz limpia y atractiva.
Funcionamiento dinámico con JavaScript.
Tecnologías Utilizadas
HTML5 → Estructura de la calculadora.
CSS3 → Diseño y estilos visuales.
JavaScript → Lógica y operaciones matemáticas.
Cómo Ejecutarlo
Descarga o clona este repositorio.
Abre el archivo index.html en tu navegador.
Usa la calculadora directamente.
Objetivo del Proyecto

Practicar el desarrollo web frontend utilizando la integración de estructura, estilos y funcionalidad en una aplicación sencilla.
